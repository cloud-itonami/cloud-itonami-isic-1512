# Contributing

This is an implemented open-business actor (maturity: **`:implemented`**).
Contributions that extend it are welcome: additional jurisdiction
spec-basis entries in `luggage.facts`, additional governor hard-checks (as
their own governed op with its own tests, following the SAME "an
independent governor re-verifies against the actor's own records before
any real-world act" pattern this repo already establishes), and operator
playbook improvements. Open an issue or PR. License: AGPL-3.0-or-later.
